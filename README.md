# CharleBin

CharleBin est une application web de type *pastebin* permettant de stocker et partager des textes de manière sécurisée et éphémère.  
Il s’agit d’un fork pédagogique du projet **PrivateBin**.

## Fonctionnalités principales

- Création de messages texte partageables via une URL
- Chiffrement côté client
- Possibilité de définir une durée d’expiration

## Pré-requis

- PHP
- Un serveur web
- Git
- Composer
